Georgia Peanut Festival — Static HTML
======================================

This is a fully static version of the festival website. To use:

1. Upload all files in this folder (keep the structure intact) to any web host:
   - GoDaddy, Bluehost, SiteGround, Netlify, Vercel, GitHub Pages, S3, etc.
2. The home page is index.html.
3. Edit any .html file in a text editor (Notepad, VS Code, etc.) to change content.
4. Replace images in the assets/ folder — keep the same filenames or update links.

Tailwind CSS is loaded via CDN (cdn.tailwindcss.com) so no build step is needed.
Google Fonts (DM Serif Display + Fira Sans) are also loaded from a CDN.

To work fully offline you would need to download Tailwind and the fonts locally.
